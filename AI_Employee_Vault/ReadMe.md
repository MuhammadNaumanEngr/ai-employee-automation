# Personal AI Employee

An autonomous AI employee that can detect tasks, plan work, request approvals, and generate reports.

## Features
- Task watcher
- AI planning using Claude
- Human approval system
- Automatic task completion
- Weekly CEO briefing

## Architecture
Watcher → Needs_Action → Claude Planning → Approval → Execution → Report

## Technologies
- Python
- Claude Code
- Obsidian Vault
- File System Automation
